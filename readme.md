# Book Now

Be first one to book your fav movie ticket from cinemaximum.

## Installation

1. python3 -m pip install --user virtualenv
2. python3 -m venv venv
3. source venv/bin/activate
4. pip install -r requirements.txt
5. export FLASK_APP=app
6. flask run
